# HTML
## 1.Tag(태그)
- 태그는<와>를 이용해 구분한다.
- 태그는 소문자로 쓴다.
- 여는 태그와 닫는 태그를 정확히 입력한다.
- 적당히 들여 쓴다.
- 태그는 속성과 함께 사용할 수 있다.
- 포함 관계를 명확히 한다.

## 2.HTML문서 기본 구조
- <!doctype html>-현재 문서과 HTML5언어로 작성된 웹문서라는 뜻.
- <html>~</html>-웹 문서의 시작과 끝을 나타내는 태그.
- <head>~</head>-웹 브라우저가 웹 문서를 해석하기 위해 필요한 정보들을 입력하는 부분. 실제 문서 내용이 아니다.
- <body>~</body>-실제로 웹 브라우저 화면에 나타날 내용.
